# AlaschgariHUD

An advanced, modular Sidekick-replica HUD for Assetto Corsa featuring floating widgets with rounded corners and custom styling options.

> [!WARNING]
> **⚠️ WICHTIGER HINWEIS / IMPORTANT NOTE**
>
> **DE:** Diese App befindet sich derzeit in der **ALPHA-Phase** und in aktiver Entwicklung. Fehler können auftreten und Funktionen können sich kurzfristig ändern.
> 
> **EN:** This app is currently in **ALPHA** stage and under active development. Bugs may occur, and features are subject to change.

---

## Features
- Modular widgets that can be moved and configured independently
- Sleek modern design with rounded corners and clean rendering
- In-game settings panel for personalization

## Installation

1. Place the `AlaschgariHUD` folder in your Assetto Corsa directory under:
   `\assettocorsa\apps\python\`
2. Enable **AlaschgariHUD** in Assetto Corsa settings or in Content Manager under Settings -> Assetto Corsa -> Apps.

## Debugging & Support

If you encounter issues or crashes, please check:
`\Documents\Assetto Corsa\logs\py_log.txt`

Or check the error log created by the app at:
`\Documents\Assetto Corsa\cfg\apps\AlaschgariHUD_error.log`
